# Japanese Trainer

Static Japanese study app for GitHub Pages.

## Features

- vocabulary training with selectable word sets;
- round-based known / unknown pools;
- local autosave plus JSON export/import;
- kana trainer with hiragana, katakana, dakuten, yoon and reverse quiz;
- textbooks and listening section linked to source PDF/audio assets.

Open `index.html` or serve the folder with any static server.
