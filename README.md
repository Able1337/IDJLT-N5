# IDJLT N5

Мобильный и настольный тренажёр японского языка уровня N5. Приложение работает как статический сайт на GitHub Pages и может быть установлено как PWA.

**Текущая версия: 0.16.9**

Сайт: https://able1337.github.io/IDJLT-N5/

## Возможности

- **Слова** — словари по урокам, объединение нескольких наборов, свайпы, стопки «Знаю / Не знаю», таблица и ромадзи.
- **Кана** — хирагана, катакана, ряды, дакутен, ёон и обратный квиз.
- **Кандзи** — отдельные знаки и сочетания, чтения кун/он, примеры, опциональные ромадзи и обратный режим для вспоминания кандзи по значению слова.
- **Фразы** — предложения и диалоги, направления русский/английский → японский и японский → русский.
- **Учебники** — просмотр PDF-учебников и аудио-плеер для `MNN I - Учебник`.
- Светлая, тёмная и OLED-темы.
- Русский и английский интерфейс.
- Полноэкранные карточки и управление свайпами.
- Сохранение настроек и прогресса в `localStorage`.
- Установка на главный экран как PWA и базовая работа из кеша.
- Прямые ссылки на отдельные режимы и словари.
- Системная кнопка «Поделиться» с копированием ссылки в браузерах без меню обмена.

## Запуск локально

Сборка не требуется. Запустите любой статический HTTP-сервер из корня репозитория:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

После этого откройте:

```text
http://127.0.0.1:4173/
```

Открывать HTML напрямую через `file://` не рекомендуется: service worker и установка PWA требуют HTTP или HTTPS.

## Структура

- `index.html` — главная страница.
- `app.js` — общая логика интерфейса и тренажёров.
- `style.css` — темы и адаптивная вёрстка.
- `data.js` — словари.
- `kanji-data.js` — кандзи и сочетания.
- `phrases-data.js` — предложения и диалоги.
- `phrases.html`, `kana.html`, `kanji.html`, `textbooks.html`, `lesson*.html` — прямые страницы режимов и уроков.
- `assets/textbooks/` — PDF-учебники.
- `assets/audio/mnn1/` — аудио к первому учебнику.
- `manifest.webmanifest`, `sw.js` — PWA и кеш.
- `VERSION` — текущая версия приложения.

## Добавление материалов

Новые наборы слов добавляются в `data.js`. Для каждой карточки нужны уникальный `id`, японское написание, перевод и при наличии ромадзи.

Новые фразы добавляются в `phrases-data.js`:

```js
{
  id: "lesson8-phrase-1",
  ru: "Русский перевод",
  en: "English translation",
  jp: "日本語の文。",
  romaji: "Nihongo no bun."
}
```

Наборы кандзи в `kanji-data.js` задаются через `sets`: наборы хранят только списки `singleIds` и `wordIds`, а описания самих кандзи остаются едиными в `singles` и `words`.

## Версионирование

Проект использует [Semantic Versioning](https://semver.org/):

- `MAJOR` — несовместимые изменения данных, прогресса или интерфейса.
- `MINOR` — новый режим, урок или заметная функция.
- `PATCH` — исправление ошибок, текста, данных или оформления.

При каждом релизе необходимо одновременно:

1. Обновить `VERSION`.
2. Обновить `APP_VERSION` и `APP_RELEASE_DATE` в `app.js`.
3. Увеличить версии `?v=` в HTML.
4. Изменить имя кеша и версии файлов в `sw.js`.
5. Дополнить раздел «История изменений» в этом README.
